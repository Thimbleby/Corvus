Form:
In this section the drop down boxes allow you to select the form of the rule for each attribute.
Identity means that this attribute's rule is 'all elements are the same'; i.e. it means that there is no rule applied to this attribute.
If the logic option is not identity, then at least one of these attributes must be assigned a non-identity form.

The item must also have no lines of symmetry running through the missing element.

In the drop down boxes below, a value of zero determines that property of that option is identical to the correct answer.
At least one property of every option listed here must be non-zero.

Form:
These four drop-down menus form the top-left four elements of the matrix.
These four elements are then used to construct the remainder of the matrix using the logic gate selected above.
All four possible combinations of the zero and one must be represented at least once in the rows and columns formed by these four elements.
The possible combinations are  (0, 0),  (0, 1),  (1, 0),  and  (1, 1).

Within the matrix, each 1 indicates the presence of the corresponding line in the test item.
While each 0 indicates the absence of that line in the test item.

The item must also have no lines of symmetry running through the missing element.
This includes diagonal lines, as well as horizontal and vertical lines.
